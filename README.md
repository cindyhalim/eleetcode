generate random problems from Leetcode to help you be more consistent at solving algo problems!

built with **Typescript powered by [Next.js](https://nextjs.org/), [Zustand](https://github.com/pmndrs/zustand), [urql](https://formidable.com/open-source/urql/)**.


</div>

### **features:**
- 🔍 filter based on topics, difficulty, and problem sets (ex: Top 100 Liked Questions) 
- ⏰ timebox problems with the help of a timer, customizable by difficulty

<br/>

### **getting started:**
1. run the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    ```
2. open [http://localhost:3000](http://localhost:3000) with your browser to see the result
